# Backup Restore Server Aplikasi di Windows Server

Berikut cara backup server aplikasi di windows server.

1. Pertama siapkan dan install software WinRar.

<figure><img src="../.gitbook/assets/image (788).png" alt=""><figcaption></figcaption></figure>

2. Misalnya kita akan backup server aplikasi Wordpress. Kita buat folder backup terlebih dahulu dengan klik kanan New > Folder masukkan nama folder.

<figure><img src="../.gitbook/assets/image (790).png" alt=""><figcaption></figcaption></figure>

3. Selanjutnya klik kanan folder wordpress > tambah ke arsip….

<figure><img src="../.gitbook/assets/image (791).png" alt=""><figcaption></figcaption></figure>

4. Lalu pada bagian Nama Arsip isi dengan backup\[tahun]\[bulan]\[tanggal].rar “backup20251201.rar” lalu klik ok.

<figure><img src="../.gitbook/assets/image (792).png" alt=""><figcaption></figcaption></figure>

5. Jika sudah, kita masukkan file nya ke dalam folder backup dan proses backup selesai.

<figure><img src="../.gitbook/assets/image (793).png" alt=""><figcaption></figcaption></figure>

Berikut cara restore backup server aplikasi di windows server 2008.

1. Siapkan file backup yang ingin di restore lalu pindahkan ke folder tujuan.

<figure><img src="../.gitbook/assets/image (794).png" alt=""><figcaption></figcaption></figure>

2. Selanjutnya klik kanan file backup nya lalu pilih ekstrak disini.

<figure><img src="../.gitbook/assets/image (795).png" alt=""><figcaption></figcaption></figure>

3. Selanjutnya kita coba apakah aplikasi berjalan atau tidak.

<figure><img src="../.gitbook/assets/image (797).png" alt=""><figcaption></figcaption></figure>
