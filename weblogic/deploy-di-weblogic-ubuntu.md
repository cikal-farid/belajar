# Deploy di WebLogic (Ubuntu)

## Deploy di WebLogic

Berikut step by stepnya :

1\.       Login terlebih dahulu di web browser, dan pastikan sudah masuk mode Lock & Edit seperti gambar dibawah ini.

<figure><img src="../.gitbook/assets/image (426).png" alt=""><figcaption></figcaption></figure>

&#x20;2\.       Pilih Deployment

<figure><img src="../.gitbook/assets/image (427).png" alt=""><figcaption></figcaption></figure>

3\.       Kemudian klik Install

<figure><img src="../.gitbook/assets/image (428).png" alt=""><figcaption></figcaption></figure>

4\.       Kemudian klik Upload You Files

<figure><img src="../.gitbook/assets/image (429).png" alt=""><figcaption></figcaption></figure>

&#x20;5\.       Kemudian klik Choose File dan pilih file yang ingin kita deploy, disini saya menggunakan sample.war dan selanjutnya klik next

<figure><img src="../.gitbook/assets/image (430).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (431).png" alt=""><figcaption></figcaption></figure>

&#x20;6\.       Dan akan muncul popup notifikasi pesan file berhasil diupload, dan klik next

<figure><img src="../.gitbook/assets/image (432).png" alt=""><figcaption></figcaption></figure>

7\.       Kemudian pilih Install this deployment as an application dan klik next

<figure><img src="../.gitbook/assets/image (433).png" alt=""><figcaption></figcaption></figure>

8\.       Cek Kembali dan jika sudah sesuai klik Finish

<figure><img src="../.gitbook/assets/image (434).png" alt=""><figcaption></figcaption></figure>

Kemudian akan muncul popup notifikasi pesan seperti gambar dibawah ini, dan lanjut klik Activate Changes

<figure><img src="../.gitbook/assets/image (435).png" alt=""><figcaption></figcaption></figure>

&#x20;9\.       Akan muncul popup notifikasi pesan perubahan telah aktif dan klik Lock & Edit

<figure><img src="../.gitbook/assets/image (436).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (437).png" alt=""><figcaption></figcaption></figure>

&#x20;10\.   Kemudian klik Control

<figure><img src="../.gitbook/assets/image (438).png" alt=""><figcaption></figcaption></figure>

&#x20;11\.   Kemudian pilih sample dan klik start dan pilih submenu Servicing all request

<figure><img src="../.gitbook/assets/image (439).png" alt=""><figcaption></figcaption></figure>

12\.   Kemudian klik Yes , maka akan muncul popup notifikasi seperti gambar dibawah ini.

<figure><img src="../.gitbook/assets/image (440).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (441).png" alt=""><figcaption></figcaption></figure>
