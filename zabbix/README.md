# Zabbix

Zabbix dibuat oleh Alexei Vladishev dan saat ini secara aktif dikembangkan dan didukung\
oleh Zabbix SIA. Zabbix merupakan enterprise class open source solusi untuk monitoring\
terdistribusi. Zabbix adalah perangkat lunak yang memantau berbagai parameter jaringan dan\
health serta inetgritas server, virtual machine, aplikasi, layanan, database, website, cloud dan lain\
lain. Zabbix menggunakan mekanisme pemberitahuan fleksibel yang memungkinkan pengguna\
untuk mengkonfirmasi peringatan berbasis email hampir semua event. Ini memungkinkan reaksi\
cepat terhadap masalah server. Zabbix menawarkan fitur pelaporan dan visualisasi data yang\
sangat baik berdasarkan data yang disimpan. Hal ini membuat Zabbix idel untuk perencanaan\
kapasitas.\
Zabbix mendukung polling dan trapping. Semua laporan Zabbix dan statistic, termasuk\
parameter konfigurasi diakses melalui front end berbasis web. Front end berbasis web memastikan\
bahwa status jaringan dan health server anda dapat dinilai dari lokasi mana pun. Dikonfigurasikan\
dengan benar, Zabbix dapat memainkan peran penting dalam memantau infrastruktur IT. Hal ini\
juga berlaku untuk organisasi kecil dengan beberapa server dan untuk perusahaan besar dengan\
banyak server.\
