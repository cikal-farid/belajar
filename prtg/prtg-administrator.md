# PRTG Administrator

Mengatur ip dan port

Disini saya ingin menambah ip nya dari yang hanya bisa di akses oleh 127.0.0.1 ini jadi 192.168.56.16:2128

1. Masuk folder PRTG dan jalankan PRTG Administrator

<figure><img src="../.gitbook/assets/image (800).png" alt=""><figcaption></figcaption></figure>

2. Di menu PRTG Web Server di isi seperti gambar dibawah ini

<figure><img src="../.gitbook/assets/image (801).png" alt=""><figcaption></figcaption></figure>

3. Di menu PRTG Core Server di isi seperti gambar dibawah ini setelah itu klik save & close

<figure><img src="../.gitbook/assets/image (804).png" alt=""><figcaption></figcaption></figure>

4. Klik OK

<figure><img src="../.gitbook/assets/image (805).png" alt=""><figcaption></figcaption></figure>

5. Proses restart dan konfigurasi perubahan sedang berjalan

<figure><img src="../.gitbook/assets/image (806).png" alt=""><figcaption></figcaption></figure>

6. Klik menu Start windows dan pilih PRTG Browser untuk masuk kembali ke dashboard PRTG atau bisa dengan membuka browser pilihan dan masukan IP dan portnya

```
http://192.168.56.17:2128
```
