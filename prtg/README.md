# PRTG

PRTG (Paessier Router Traffic Grapher) Network Monitor adalah perangkat lunak yang mudah digunakan untuk memantau penggunaan bandwith dan banyak parameter jaringan lain melalui SNMP, Packet Sniffing, atau Cisco NetFlow yang memungkinkan untuk pengukuran traffic berdasarkan alamat ip atau protocol. PRTG dapat memonitoring ketersediaan jaringan dan pemakaian jaringan menggunakan berbagai macam protocol, meliputi SNMP, Netflow, dan WMI. Aplikasi ini termasuk powerful, menawarkana tampilan antarmuka berbasis website yang mudah digunakan serta aplikasi untuk iOS dan Android.

Berikut cara memasang PRTG Monitoring System.
