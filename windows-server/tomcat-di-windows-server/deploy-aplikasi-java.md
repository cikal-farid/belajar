# Deploy Aplikasi Java

INSTALL APLIKASI JAVA DENGAN TOMCAT DI WINDOWS\
Berikut cara installasi atau deploy aplikasi java dengan tomcat di windows.

1. Buka web browser masukkan alamat ip : \[port] seperti berikut.

```
http://192.168.56.16:8082/
```

<figure><img src="../../.gitbook/assets/image (157).png" alt=""><figcaption></figcaption></figure>

2. Selanjutnya klik Manager App dan login dengan user yang telah dibuat sebelumnya.

<figure><img src="../../.gitbook/assets/image (158).png" alt=""><figcaption></figcaption></figure>

3. login terlebih dahulu (cikal)

<figure><img src="../../.gitbook/assets/image (160).png" alt=""><figcaption></figcaption></figure>

4. Selanjutnya scroll ke bagian Deploy lalu di bagian War file to deploy klik tombol choose file.

<figure><img src="../../.gitbook/assets/image (161).png" alt=""><figcaption></figcaption></figure>

5. Pilih file yang akan di deploy lalu klik Open.

<figure><img src="../../.gitbook/assets/image (163).png" alt=""><figcaption></figcaption></figure>

6. Lalu klik Deploy.

<figure><img src="../../.gitbook/assets/image (164).png" alt=""><figcaption></figcaption></figure>

7. Jika sudah, cari di bagian Applications / \[nama file deploy]. Lalu klik kanan > open in new tab.

<figure><img src="../../.gitbook/assets/image (165).png" alt=""><figcaption></figcaption></figure>

8. Jika muncul maka file berhasil di deploy.

<figure><img src="../../.gitbook/assets/image (166).png" alt=""><figcaption></figcaption></figure>
