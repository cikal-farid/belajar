# Deploy Aplikasi Java

Install Aplikasi Java dengan WebLogic di Windows Berikut cara install aplikasi Java dengan WebLogic di Windows Server 2008 :

1. Login ke WebLogic terlebih dahulu, pastikan sudah masuk ke menu Deployment

<figure><img src="../../.gitbook/assets/image (103).png" alt=""><figcaption></figcaption></figure>

2. Kemudian klik Install dan klik upload your files

<figure><img src="../../.gitbook/assets/image (104).png" alt=""><figcaption></figcaption></figure>

3. Lalu lanjut pilih Choose pada bagian Deployment Archive

<figure><img src="../../.gitbook/assets/image (105).png" alt=""><figcaption></figcaption></figure>

4. Pilih project yang ingin di deploy

<figure><img src="../../.gitbook/assets/image (106).png" alt=""><figcaption></figcaption></figure>

5. Kemudian klik next

<figure><img src="../../.gitbook/assets/image (107).png" alt=""><figcaption></figcaption></figure>

6. Setelah klik next sebelumnya dan akan muncul notifikasi seperti gambar dibawah ini dan lanjut klik next

<figure><img src="../../.gitbook/assets/image (108).png" alt=""><figcaption></figcaption></figure>

7. Pilih install this deployment as an application dan next

<figure><img src="../../.gitbook/assets/image (109).png" alt=""><figcaption></figcaption></figure>

8. Lanjut klik next

<figure><img src="../../.gitbook/assets/image (111).png" alt=""><figcaption></figcaption></figure>

9. Klik Save

<figure><img src="../../.gitbook/assets/image (112).png" alt=""><figcaption></figcaption></figure>

10. Selanjutnya kita cek kembali dengan klik Deployments > Control > Centang aplikasi yang sudah di deploy > Start > Servicing all requests.

<figure><img src="../../.gitbook/assets/image (114).png" alt=""><figcaption></figcaption></figure>

10. Klik seperti pada bagian di gambar bawah ini untuk melihat hasil deploy nya

<figure><img src="../../.gitbook/assets/image (113).png" alt=""><figcaption></figcaption></figure>

11. Jika berhasil terbuka maka deploy telah sukses di lakukan.

<figure><img src="../../.gitbook/assets/image (115).png" alt=""><figcaption></figcaption></figure>
