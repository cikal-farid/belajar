# VMWare

VM atau Virtual Machine merupakan software yang digunakan untuk virtualisasi OS pada OS dan perangkat keras yang berjalan atau digunakan. Fungsi utama dari VM adalah untuk melakukan tugas-tugas yang tidak bisa dikerjakan pada OS yang berjalan. Contoh software VM yaitu VMWare. Kita dapat mengunduh VMWare pada situs [https://www.vmware.com/products/workstation-player.html](https://www.vmware.com/products/workstation-player.html).

1. Download VMWare melalui situs dibawah ini

```
https://www.vmware.com/products/workstation-pro.html
```
